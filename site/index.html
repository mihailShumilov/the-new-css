The New CSS

A super lightweight, utility-first CSS library. Modern CSS features, familiar class names, no build step required.

Get Started GitHub
<link rel="stylesheet" href="https://unpkg.com/the-new-css/dist/the-new-css.min.css">

Why The New CSS?

✨

Modern CSS

Built with oklch() colors, @layer, container queries, :has(), and CSS nesting.

⚡

Under 5KB

Full library gzipped to under 5KB. Core bundle even smaller at ~4.5KB.

🛠

No Build Step

Just add a <link> tag. Works with any stack - HTML, React, Vue, Svelte, anything.

🎨

Customizable

Override CSS custom properties to match your brand. Design tokens make it easy.

📦

Modular

Import only what you need. Each category is available as a separate module.

📱

Responsive

Mobile-first responsive variants at sm, md, and lg breakpoints.

Quick Start

1. Add the stylesheet

<link rel="stylesheet" href="https://unpkg.com/the-new-css/dist/the-new-css.min.css">

2. Or install via npm

npm install the-new-css

3. Start building

<div class="flex items-center gap-4 p-6 bg-white rounded-lg shadow"> <h2 class="text-xl font-bold">Hello World</h2> <p class="text-gray-600">Built with The New CSS</p> </div>
